# Moodit

사용자의 감정과 소비 데이터를 기반으로 소비 행동 패턴을 분석하는 AI 소비 심리 분석 모바일 애플리케이션입니다.

## 프로젝트 소개

Moodit은 사용자가 기록하고 누적한 일일 소비 데이터(카테고리별 누적 금액, 소비 이유, 빈도, 총 금액)를 분석하여 소비 심리 유형을 도출하는 모바일 앱입니다. 

기존의 단순 1회성 금액대 구간 선택 입력 방식을 넘어서서, 사용자가 직접 소비 금액을 숫자로 상세하게 입력하고 하루 동안의 내역을 목록 형태로 관리하여 전체 데이터를 기반으로 종합적인 AI 분석 및 시각화 차트를 제공받을 수 있도록 구현되었습니다.

## 기술 스택

* **언어**: Kotlin
* **UI**: Jetpack Compose (Material3 디자인 가이드라인 준수)
* **네비게이션**: Navigation Compose
* **AI 엔진**: GroqCloud API (Llama 3.3 모델 연동)
* **로컬 스토리지**: Preferences DataStore (최근 분석 정보 로컬 저장)
* **스플래시**: AndroidX Core SplashScreen API

## 주요 기능

* **📍 위치 정보 및 알림 연동**: 사용자 GPS 위치 정보를 조회하여 소비 주소를 한글로 변환하고, 조건형 알림(늦은 시간 소비 알림 등)을 발송합니다.
* **📝 다건 소비 내역 기록 및 누적 관리**:
  * 금액을 텍스트 필드로 직접 입력하고 숫자 키패드를 지원합니다.
  - 소비 이유 선택 시 카드형 버튼 UI를 도입하여 이모지와 텍스트를 함께 제공하고 일관성 있는 디자인을 유지합니다.
  - '오늘의 소비 내역' 영역에서 임시 저장된 여러 건의 기록 목록(`LazyColumn`)을 한눈에 보고 실시간 수정 및 삭제를 진행할 수 있습니다.
  - 소비가 변동될 때마다 전체 총 소비 금액이 실시간으로 합산 및 자동 포맷팅되어 표시됩니다.
* **📊 Canvas 기반 소비 시각화 차트**:
  - 결과 화면에서 카테고리별 누적 소비 비율을 부채꼴 각도로 계산해 그려주는 커스텀 **파이 차트(Pie Chart)**를 제공합니다.
  - 각 카테고리별 소비 금액 규모를 다른 항목과 쉽게 비교할 수 있도록 커스텀 **막대 그래프(Bar Chart)**를 렌더링합니다.
* **🤖 AI 소비 성향 분석 리포트**:
  - GroqCloud API를 활용해 한국어 명사 형태의 명확한 소비 유형(예: '자기보상형 소비자')과 특징을 자연스러운 한글로 설명하는 상세 리포트를 제공합니다 (한자, 일본어, 불필요한 영어 배제).
* **✨ 실시간 최근 소비 인사이트 카드**:
  - 이전 분석 결과의 조언 부분에서 한글 문장 길이 **40~60자 내외**의 친근하고 부드러운 말투(`~해 보세요`, `~이어가 보세요`, `~유지해 보세요`, `~어떨까요?`)를 가진 실질적이고 개인화된 인사이트 핵심 1문장을 추출하여 홈 화면 상단 카드에 말줄임 없이 실시간 연동해 줍니다.
  - 가독성을 높이기 위해 인사이트 카드 내 캐릭터를 제거하고 오로지 텍스트 영역에 집중하게 레이아웃을 확장했습니다.
* **🔙 메인화면으로 돌아가기**:
  - 결과 화면의 맨 아래에 기존 분석 버튼과 동일한 스타일(높이, 둥글기, 그림자, 스케일 반응 효과)의 '메인화면으로 돌아가기' 버튼을 배치했습니다.
  - 복귀 시 백스택을 정리(`popUpTo("main") { inclusive = true }`)하여, 홈으로 돌아간 후 뒤로가기를 눌렀을 때 불필요한 이전 결과/로딩 페이지로의 재진입을 차단합니다.
* **🚀 브랜드 스플래시 화면 적용**:
  - Android 12 이상 플랫폼 요구사항에 최적화된 스플래시 화면을 제공합니다.
  - 원형 마스크에 의해 캐릭터 얼굴이 잘리지 않도록, 사방(특히 하단부)에 넉넉한 여백 패딩을 적용한 스플래시 전용 이미지(`today_character_splash.png`)를 분리해 안전하게 렌더링합니다.

## 앱 동작 흐름 (Application Flow)

```
[메인 화면 (MainScreen)]
  - 상단 인사말 영역 (우측 끝에 65dp 캐릭터 정렬 배치 및 20dp 여백)
  - 최근 소비 인사이트 카드 (캐릭터 없이 텍스트 전면 배치, 글자 수 40~60자)
  - 최근 분석 정보 복원 노출 (DataStore 연동)
   ↓ (새 소비 분석하기 클릭)
[소비 입력 화면 (InputScreen)]
  - 금액 직접 입력, 이모지 결합 소비 이유 카드 선택, 메모 작성
  - '소비 저장하기'로 임시 누적 (LazyColumn 목록에서 수정/삭제 기능)
   ↓ (소비 분석하기 클릭 - 최소 1건 이상 등록 필요)
[로딩 화면 (LoadingScreen)]
  - AI 분석 중 상태 애니메이션 표시
   ↓ (Groq API 호출 및 결과 반환)
[결과 화면 (ResultScreen)]
  - 소비 유형, 시각화 차트(Pie/Bar), AI 리포트, 메모 모음 및 위치 정보 확인
  - DataStore에 분석 결과 및 40~60자 인사이트 즉시 영구 백업
   ↓ (메인화면으로 돌아가기 클릭 - 백스택 popUpTo 정리)
[메인 화면 (MainScreen)]
  - 새로 갱신된 최근 소비 인사이트 및 유형 데이터 실시간 적용 노출
```

## 실행 및 환경 설정

* Groq API Key는 `local.properties` 파일에 아래와 같이 등록하여 사용합니다.
  ```properties
  GROQ_API_KEY=YOUR_API_KEY_HERE
  ```
