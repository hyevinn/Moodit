# F-02 홈 화면 제공 기능

## 설명

시스템은 서비스 소개, 최근 소비 분석 결과 요약 표시 및 분석 시작 화면을 제공한다. 최근 소비 분석 결과는 Preferences DataStore에 저장된 데이터를 복원해 실시간으로 표시하며, 앱 재실행 간에도 정보가 유지된다.

## Acceptance Criteria

- [ ] 홈 화면은 서비스 소개 및 분석 시작 화면을 제공한다.
- [ ] 홈 화면에서 '소비 기록하기' 버튼을 클릭하면 소비 입력 화면으로 이동한다.
- [ ] 홈 화면의 '최근 분석 결과' 영역에 가장 최근 수행한 소비 분석 유형과 그 설명을 표시한다.
- [ ] 홈 화면의 '최근 선택한 소비 정보' 영역에 가장 최근 분석 시 선택했던 카테고리, 금액대, 소비 이유 정보를 표시한다.
- [ ] 로컬에 저장된 이전 분석 결과가 존재하지 않는 경우(초기 진입 시), 기본 가이드라인 문구 및 기본 이모지("🎁")를 표시한다.
- [ ] 로컬에 저장된 이전 분석 결과가 존재할 경우, 저장된 소비 이유 유형에 따라 동적으로 좌측 이모지를 매핑하여 표시한다 ("자기만족" -> "🎁", "스트레스 해소" -> "💨", "유행 영향" -> "✨", 기타 -> "✔").
- [ ] 앱을 완전히 종료하고 다시 실행해도 DataStore를 통해 복원된 최신 분석 결과가 유지되어 화면에 표시된다.
