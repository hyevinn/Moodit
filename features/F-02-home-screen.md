# F-02 홈 화면 제공 기능

## 설명

시스템은 서비스 소개, 최근 소비 분석 결과 및 소비 인사이트 요약 표시, 그리고 분석 시작 화면을 제공한다. 최근 소비 분석 결과와 인사이트 문장은 Preferences DataStore에 저장된 데이터를 복원해 실시간으로 표시하며, 앱 재실행 간에도 정보가 유지된다.

## Acceptance Criteria

- [ ] 홈 화면은 서비스 소개 및 분석 시작 화면을 제공한다.
- [ ] 홈 화면에서 '새 소비 분석하기' 버튼을 클릭하면 소비 입력 화면으로 이동한다.
- [ ] 홈 화면 상단의 인사말 영역은 다음과 같이 구조화되어 표시된다.
  - **첫 번째 줄**: "안녕하세요 👋" 텍스트만 단독 노출.
  - **두 번째 줄**: `Row` (`Modifier.fillMaxWidth()`, `Arrangement.SpaceBetween`, `Alignment.CenterVertically`)를 사용해 왼쪽에는 "오늘도 현명한 소비 습관을 만들어봐요" 서브 텍스트, 오른쪽에는 `today_character` 구름 캐릭터(`65.dp`)를 정렬 배치.
- [ ] 상단 인사말 `Row`와 하단의 인사이트 카드 사이에는 시각적 안정을 위해 충분한 여백(`20.dp` Spacer)이 확보된다.
- [ ] 홈 화면 중앙에는 '✨ 최근 소비 인사이트' 카드를 배치한다.
  - 인사이트 텍스트 영역을 넓혀 자연스러운 줄바꿈을 유도하기 위해 카드 내 캐릭터 이미지를 제외하고 **인사이트 제목과 문구만 단독 노출**한다.
  - 제목은 `18sp Bold` 검은색 스타일, 본문 내용(`displayInsight`)은 `15sp Normal` 검은색 스타일로 줄바꿈 및 말줄임(...) 없이 표시한다.
  - 카드의 세로 높이는 텍스트 길이에 따라 자동으로 증가할 수 있도록 `heightIn(min = 140.dp)`을 적용한다.
- [ ] 홈 화면의 '최근 분석 결과' 영역에 가장 최근 수행한 소비 분석 유형(예: '자기보상형 소비')과 그 설명을 표시한다.
- [ ] 홈 화면의 '최근 선택한 소비 정보' 영역에 가장 최근 분석 시 입력했던 카테고리, 금액대, 소비 이유 정보를 표시한다.
- [ ] 로컬에 저장된 이전 분석 결과 및 최근 소비 인사이트가 존재하지 않는 경우(초기 진입 시), 기본 가이드라인 문구("오늘의 소비에는 오늘의 감정이 담겨 있어요.") 및 기본 이모지("🎁")를 표시한다.
- [ ] 로컬에 저장된 이전 분석 결과가 존재할 경우, 저장된 소비 이유 유형에 따라 동적으로 좌측 이모지를 매핑하여 표시한다 ("자기만족" -> "🎁", "기분전환" -> "💨", "유행 영향" -> "✨", 기타 -> "✔").
- [ ] 앱을 완전히 종료하고 다시 실행해도 DataStore를 통해 복원된 최신 분석 결과와 최근 소비 인사이트가 유지되어 화면에 표시된다.
