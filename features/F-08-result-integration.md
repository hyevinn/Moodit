# F-08 결과 화면 통합 제공 기능

## 설명

시스템은 소비 분석 결과를 하나의 화면에 통합하여 제공하며, 분석 완료된 최신 통찰 결과를 로컬 Preferences DataStore에 캐싱하여 홈 화면에 즉시 복원 연동하는 데이터 파이프라인을 지원한다.

## Acceptance Criteria

- [ ] 소비 분석 결과 화면(`ResultScreen`)은 스크롤 가능한 구조로 구성되며 다음 카드 및 컴포넌트들을 차례대로 레이아웃에 포함한다:
  1. **소비 유형 결과 카드**: AI 분석을 통해 도출된 소비 유형 명칭(예: '자기보상형 소비') 및 요약 설명 제공.
  2. **AI 소비 분석 리포트 카드**: 생성된 상세 리포트 텍스트 제공.
  3. **소비 분석 시각화 카드**: 카테고리별 누적 비율 파이 차트와 소비 금액 규모를 보여주는 막대 그래프 영역 제공.
  4. **소비 성향 키워드 카드**: 태그 형태의 세 가지 성향 키워드 표시.
  5. **입력한 소비 정보 요약 카드**: 주요 카테고리, 총 소비 금액, 주요 소비 이유, 수집된 소비 위치 정보를 결합해 요약 정보 표시.
  6. **오늘의 소비 한마디 모음 카드**: 사용자가 오늘 입력한 개별 메모들이 존재할 경우 쉼표로 연결해 한곳에 통합 모음 제공.
  7. **메인화면으로 돌아가기 버튼**: 뒤로가기 스택을 팝업 청소하며 홈 화면으로 이동하는 버튼 제공.
- [ ] [통합 데이터 연동 흐름]:
  - `ResultScreen` 진입 -> AI 리포트 획득 -> 40~60자의 한 줄 조언(인사이트) 및 소비 분석 데이터 추출 -> `DataStoreManager`를 사용해 DataStore에 즉시 영구 저장 -> 홈 화면(`MainScreen`)으로 돌아왔을 때 해당 데이터를 읽어와 실시간으로 인사이트 카드 및 최근 분석 정보 카드에 지연 없이 반영.
- [ ] 앱이 완전히 종료된 후 다시 켜져도 DataStore에 보관되어 있던 이전 최종 인사이트 및 분석 결과를 홈 화면에 끊김 없이 복원한다.
