# F-08 결과 화면 통합 제공 기능

## 설명

시스템은 소비 분석 결과를 하나의 화면에서 제공하며, 이를 바탕으로 결과 화면에서 추출된 핵심 문장을 로컬에 보관하고 메인 화면에 연동해 주는 통합 데이터 파이프라인을 지원한다.

## Acceptance Criteria

- [ ] 소비 정보 카드가 표시된다.
- [ ] 소비 유형 카드가 표시된다.
- [ ] 소비 성향 태그가 표시된다.
- [ ] AI 분석 리포트가 표시된다.
- [ ] [통합 데이터 연동 흐름]: 결과 화면 (ResultScreen) 생성 -> AI 소비 리포트 로드 -> 핵심 문장 추출 -> `DataStoreManager`를 통해 DataStore에 저장 -> 메인 화면 (MainScreen) 상단 인사이트 카드로 전달 및 실시간 노출의 데이터 흐름이 정상 작동한다.
- [ ] 앱이 강제 종료되거나 재부팅된 후에도 DataStore에 보관되어 있던 최신 인사이트 조언 문장을 읽어와 메인 화면에 끊김 없이 유지 및 복원한다.
