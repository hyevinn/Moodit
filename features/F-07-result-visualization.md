# F-07 결과 화면 시각화 기능

## 설명

시스템은 저장된 전체 소비 내역을 기반으로 소비 분석 결과를 시각적 차트 및 수치로 제공한다.

## Acceptance Criteria

- [ ] 사용자가 입력하고 누적한 당일 전체 소비 데이터를 카테고리별로 자동 분류하여 집계한다.
- [ ] **📊 카테고리별 소비 비율 (파이 차트)**:
  - Jetpack Compose `Canvas` API를 활용하여 각 카테고리별 누적 금액 비율에 따른 부채꼴 각도(`drawArc`)를 계산해 커스텀 파이 차트(Pie Chart)를 그린다.
  - 우측에 각 카테고리명, 백분율(%), 그리고 포맷팅된 카테고리별 금액을 색상 칩과 함께 목록형 범례로 함께 제공한다.
- [ ] **📈 카테고리별 소비 금액 (막대 그래프)**:
  - 각 카테고리별 총 소비 금액 규모를 다른 카테고리와 비교할 수 있도록 높이가 다르게 표시되는 커스텀 막대 그래프(Bar Chart)를 렌더링한다.
  - 막대 위에 해당 카테고리의 금액 정보(만 단위 등 축약 표현 적용)를 출력하고, 하단에 카테고리 명을 함께 정렬하여 표시한다.
- [ ] 차트에 쓰이는 색상은 카테고리별로 고유한 테마 색상(식비: 붉은 계열, 카페: 황색 계열, 교통: 청색 계열 등)을 일관되게 매핑하여 가독성을 높인다.
