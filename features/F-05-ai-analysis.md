# F-05 AI 소비 성향 분석 기능

## 설명

시스템은 사용자의 소비 정보를 기반으로 소비 유형과 소비 성향을 분석한다. 이 과정에서 한글 정제 가이드라인을 명시해 분석 결과를 가독성 높은 자연스러운 한국어로 도출하며, 생성된 리포트 내용으로부터 핵심 인사이트 문장을 오프라인 상에서 요약/추출한다.

## Acceptance Criteria

- [ ] 소비 카테고리와 소비 이유를 분석에 활용한다.
- [ ] GroqCloud API를 활용하여 소비 성향 리포트를 생성한다.
- [ ] 소비 특징과 소비 습관을 자연어 형태로 생성한다.
- [ ] AI 분석 결과는 한자(Hanja), 일본어 및 불필요한 외국어 표현을 완전히 배제하고 오직 자연스러운 한국어로만 생성한다.
- [ ] 분석 결과 중 [소비 유형]은 '필요형 소비자', '자기보상형 소비자', '실용중심형 소비'와 같이 한자가 포함되지 않은 완전한 한국어 명사 형태롤 도출한다.
- [ ] 생성 완료 시, 문장이 여러 개이고 글자 수가 60자를 초과하는 경우 첫 번째 핵심 조언 문장만을 추출 및 요약(`cleanAndSummarize`)하여 최근 소비 인사이트 데이터로 활용한다.
