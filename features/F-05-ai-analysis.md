# F-05 AI 소비 성향 분석 기능

## 설명

시스템은 사용자가 저장한 다건의 소비 내역(카테고리별 누적 금액, 소비 이유, 소비 빈도, 총 소비 금액)을 기반으로 소비 유형과 소비 성향을 종합적으로 분석한다. 분석 결과 리포트는 철저한 정제 규칙에 따라 생성되며, 메인 화면용 '최근 소비 인사이트' 문구를 엄격한 조건(글자 수, 어조, 문장 구성) 하에 한 문장으로 제공한다.

## Acceptance Criteria

- [ ] 단일 소비 데이터 분석 대신, 당일 저장된 모든 소비 내역 목록 전체를 분석 매개변수로 활용하여 카테고리별 비중 및 소비 습관을 분석한다.
- [ ] GroqCloud API(Llama 3.3 모델)를 활용하여 사용자 맞춤형 소비 성향 분석 데이터를 획득한다.
- [ ] AI 분석 텍스트 생성 시 한국어 외 한자(Hanja), 일본어 및 불필요한 영어 표현 사용을 금지하며 가독성이 높은 자연스러운 한국어 문장만을 생성한다.
- [ ] 분석 결과 항목 중 [소비 유형]은 '필요형 소비자', '자기보상형 소비자', '실용중심형 소비자' 등과 같이 한자가 포함되지 않은 완전한 한국어 명사 형태로만 도출한다.
- [ ] 분석 완료 시 생성되는 [한 줄 조언](최근 소비 인사이트)은 다음 조건을 엄격히 준수하여 1문장으로 생성 및 추출한다.
  - **자연스러운 표현**: 사용자가 바로 이해할 수 있는 자연스러운 한국어 문장이어야 한다.
  - **유기적인 결합**: 조언들을 나열하는 표현이 아닌, 하나의 유기적인 흐름으로 이어지는 문장이어야 한다.
  - **글자 수 범위**: 화면 줄바꿈 시 가독성을 해치지 않도록 공백을 포함하여 **40~60자 내외**로 작성되어야 한다.
  - **부드럽고 친근한 어조**: `~노력해요`, `~하도록 하세요`와 같은 딱딱한 지시형 표현 대신 `~해 보세요`, `~이어가 보세요`, `~유지해 보세요`, `~어떨까요?` 등의 친근한 어미를 사용하여 조언해야 한다.
  - **실질적이고 개인화된 조언**: 단순 고정 문구 또는 지나치게 뻔한 잔소리가 아니라, 실제 사용자의 지출 내역(금액, 카테고리, 소비 이유 등)에 어울리는 구체적이고 실질적인 통찰력을 1문장으로 담아야 한다.
