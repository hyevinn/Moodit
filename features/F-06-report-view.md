# F-06 소비 리포트 제공 기능

## 설명

사용자는 AI가 생성한 소비 성향 리포트를 확인하고 분석 완료 즉시 Preferences DataStore를 통해 로컬에 저장한다. 또한 사용자가 리포트를 모두 확인한 뒤 홈 화면으로 돌아갈 수 있도록 하단에 네비게이션 버튼을 제공한다.

## Acceptance Criteria

- [ ] 분석 결과가 텍스트 형태로 출력되며 소비 특징 및 소비 습관 설명이 포함된다.
- [ ] 소비 분석 결과가 화면에 노출되는 시점(ResultScreen 구성 시)에 최근 분석 데이터(소비 유형, 설명, 카테고리, 금액대, 소비 이유) 및 추출된 최근 소비 인사이트가 Preferences DataStore에 영구 저장된다.
- [ ] 결과 화면의 가장 하단(소비 결과 및 그래프 영역 아래)에 **'메인화면으로 돌아가기'** 버튼을 배치한다.
  - 이 버튼의 크기(높이 60.dp), 색상(Primary 색상 및 틴팅 피드백), 모서리 둥글기(18.dp), 그림자 및 꾹 눌림 애니메이션(0.96f 스케일 모션) 등 전체적인 스타일은 입력 화면의 '소비 분석하기' 버튼과 동일한 디자인으로 구현되어 일관성을 유지한다.
- [ ] '메인화면으로 돌아가기' 버튼을 클릭하면 앱의 메인 화면(라우트 `"main"`)으로 네비게이션이 동작한다.
- [ ] 메인으로 복귀 시 뒤로가기 스택이 불필요하게 쌓이지 않도록 팝업 설정을 진행한다.
  - `popUpTo("main") { inclusive = true }` 및 `launchSingleTop = true` 옵션을 사용해 스택을 청소하며, 메인으로 복귀한 뒤 기기 뒤로가기 버튼을 눌렀을 때 이전 분석 결과 화면이나 로딩 화면으로 다시 이동하지 않고 정상 종료/퇴장 처리되도록 구성한다.
