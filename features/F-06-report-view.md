# F-06 소비 리포트 제공 기능

## 설명

사용자는 AI가 생성한 소비 성향 리포트를 확인할 수 있으며, 결과가 노출되는 즉시 최근 분석 결과가 Preferences DataStore를 통해 로컬에 저장된다.

## Acceptance Criteria

- [ ] 분석 결과가 텍스트 형태로 출력된다.
- [ ] 소비 특징 및 소비 습관 설명이 포함된다.
- [ ] 사용자는 소비 분석 결과를 화면에서 확인할 수 있다.
- [ ] 소비 분석 결과가 화면에 노출되는 시점(ResultScreen 구성 시)에 최근 분석 데이터(소비 유형, 설명, 카테고리, 금액대, 소비 이유)가 Preferences DataStore에 영구 저장된다.
