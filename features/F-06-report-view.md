# F-06 소비 리포트 제공 기능

## 설명

사용자는 AI가 생성한 소비 성향 리포트를 확인할 수 있으며, 결과가 노출되는 즉시 최근 분석 결과가 Preferences DataStore를 통해 로컬에 저장된다. 결과 화면의 AI 리포트는 홈 화면 상단 '✨ 최근 소비 인사이트'의 데이터 원본 역할을 담당한다.

## Acceptance Criteria

- [ ] 분석 결과가 텍스트 형태로 출력된다.
- [ ] 소비 특징 및 소비 습관 설명이 포함된다.
- [ ] 사용자는 소비 분석 결과를 화면에서 확인할 수 있다.
- [ ] 소비 분석 결과가 화면에 노출되는 시점(ResultScreen 구성 시)에 최근 분석 데이터(소비 유형, 설명, 카테고리, 금액대, 소비 이유) 및 추출된 핵심 인사이트 문장이 Preferences DataStore에 영구 저장된다.
- [ ] 추가 API 호출을 절대 발생시키지 않고, 결과 화면에서 이미 생성된 AI 리포트 내 텍스트 조언 부분을 온전히 재활용하여 인사이트 카드를 갱신한다.
