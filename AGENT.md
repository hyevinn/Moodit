# Moodit

## 프로젝트 소개
Moodit은 사용자의 소비 정보를 기반으로 소비 유형과 소비 성향을 분석하는 AI 기반 소비 분석 앱이다.

## 기술 스택
- Kotlin
- Jetpack Compose
- Material3
- Navigation Compose
- GroqCloud API

## 현재 구현 기능

### FR-00 위치 정보 수집 기능
- 위치 권한 요청
- 위치 정보 확인

### FR-01 알림 수신 동의 기능
- 알림 권한 요청
- 알림 수신 동의
- 알림 수신 기능

### FR-02 홈 화면 제공 기능
- 서비스 소개 화면 제공
- 소비 기록하기 버튼 제공

### FR-03 선택형 소비 입력
- 소비 카테고리 선택
- 소비 금액대 선택
- 소비 이유 선택

### FR-04 ~ FR-05 소비 성향 분석
- GroqCloud API 호출
- 소비 성향 분석
- 로딩 상태 표시

### FR-06 ~ FR-08 결과 리포트 제공
- 소비 정보 카드
- 소비 유형 카드
- 소비 성향 태그
- AI 분석 리포트

### FR-09 테마 전환 기능
- 다크 모드
- 라이트 모드
- 테마 상태 유지

## 최근 UI 개선
- 최근 선택한 소비 정보 카드 추가
- 소비 정보 태그(Chip) 디자인 개선
- 버튼 클릭 애니메이션 추가
- 메인 화면 UI 개선

## 개발 규칙
- Jetpack Compose 사용
- Material3 사용
- API Key 하드코딩 금지
- BuildConfig 사용
